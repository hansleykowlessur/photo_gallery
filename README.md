## Run Locally

Clone the project

```bash
  git clone https://github.com/hansleykowlessur/photo_gallery.git
```

Go to the project directory

```bash
  cd photo_gallery
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
